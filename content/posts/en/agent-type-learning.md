# Learning Agents

Learning agents adapt from data, outcomes, and feedback loops.

## Best use cases

- Dynamic environments
- Evolving user behavior
- Long-horizon optimization

## Trade-offs

- Need quality feedback
- Can drift if reward signals are poor
