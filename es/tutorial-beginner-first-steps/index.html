Carril principiante — Tu primer agente de IA (sin base técnica)

Tutorial Tutorial Dificultad: beginner Beginner No-Code Ollama Flowise
Carril principiante: no se requiere formación técnica.

Para qué es este carril

El carril principiante es para personas sin formación técnica que quieren entender en la práctica qué se siente un «agente de IA».

Estado: contenido provisional. Aquí irán tutoriales no técnicos (capturas, listas de comprobación y pasos de copiar y pegar).

Objetivos de aprendizaje (borrador)

Cuando el carril esté completo, el lector debería poder:

  • Explicar con palabras sencillas qué es una tarea para un agente frente a un chat normal
  • Seguir un flujo visual (cajas y flechas) sin leer código
  • Ejecutar un modelo local con instaladores guiados (idea: «IA en tu ordenador»)
  • Distinguir respuestas de un solo paso y comportamiento multipaso

Esquema de módulos (marcador)

Módulo 1 — Mentalidad

  • Qué es un agente en una frase
  • Por qué importa más «paso a paso» que «modelo más listo»

Módulo 2 — Tu primer flujo (sin código)

  • Marcador: conectar bloque de prompt → modelo local → respuesta
  • Marcador: tres tareas de ejemplo (plan de viaje, esquema de artículo, resumen de notas)

Módulo 3 — Que sea útil

  • Marcador: mejorar instrucciones sin tocar código
  • Marcador: guardar y reutilizar la configuración

Módulo 4 — Seguridad y límites

  • Marcador: qué no pegar en herramientas
  • Marcador: cuándo pedir ayuda a una persona

Siguiente paso

Abre el tutorial destacado en el hub de tutoriales — Construye tu primer agente de IA (2 rutas gratuitas) — y usa la Parte A (sin código) como columna vertebral técnica mientras este carril gana narrativa y visuales.
